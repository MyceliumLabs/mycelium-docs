## Testnet

### MetaMask

To add Mycelium Testnet as a custom network to MetaMask:

1. Open the MetaMask browser extension.
2. Open the network selection dropdown by clicking the dropdown button at the top of the extension.
3. Click the **Add network** button.
4. Click **Add a network manually**.
5. In the **Add a network manually** dialog that appears, enter the following information:

| **Name**         | **Value**                                     |
|------------------|-----------------------------------------------|
| Network Name     | Mycelium Testnet                              |
| RPC Endpoint     | `https://testnet.myceliumlabs.xyz`                |
| Chain ID         | `15551`                                       |
| Currency Symbol  | `ECO`                                         |
| Block Explorer   | [https://testnet.explorer.myceliumlabs.xyz](https://testnet.explorer.myceliumlabs.xyz) |

6. Tap the **Save** button to save Mycelium Testnet as a network.

You should now be able to connect to the Mycelium Testnet by selecting it from the network selection dropdown.
